# Portfolio Lab - 资产回测平台评估与 TODO

> 评估时间：2026-03-31
> 评估方式：代码分析 + 浏览器实际访问 (http://127.0.0.1:3000)

---

## 一、当前状态总结

### 已有的东西
- Next.js 前端 + FastAPI 后端 + 后台 Worker 三件套
- 核心回测引擎（固定权重、每日模拟、多币种 FX 转换、再平衡）
- 资产选择器（A股/美股/港股/加密四个市场 tab）
- 任务队列（File / Redis 后端可切换）
- 任务详情页（指标卡片 + SVG 净值曲线 + 回撤曲线）
- 中文本地化

### 核心问题
项目目前更像一个**回测引擎的技术 demo**，离真正可用的「资产回测平台」差距很大。主要体现在：

1. **功能链路跑不通** - akshare 安装失败、A股搜索返回空、结束日期必须是周五等，新用户进来基本无法完成一次完整回测
2. **功能单薄** - 只有"提交任务 → 看结果"一条路，没有策略模板、没有历史对比、没有多组合对比
3. **交互体验粗糙** - 纯表单提交式交互，图表是基础 SVG 手绘，没有 tooltip/缩放/时间轴交互
4. **没有数据沉淀** - 每次都要实时拉数据生成快照，没有本地数据管理
